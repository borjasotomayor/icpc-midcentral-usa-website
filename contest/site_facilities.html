Tennessee Technological University¶

Site director: Martha Kosa

Operating system: Windows 10

Software Environment

  • Supported Languages:
    • C (using the latest MinGW gcc)
    • C++ (using the latest MinGW g++)
    • Java (using Java JDK 8u181)
    • Python 2 (using PyPy 6.0.0 for Python 2.7.13)
    • Python 3 (using CPython 3.6.5)
  • Resources
    • Test-Compilers folder on the desktop with script to test compiling/running in all supported languages (using the options Kattis uses).
    • Documentation Folder on the Desktop with offline documentation for all supported languages.
    • Info.pdf document on the Desktop explaining the resources, compiler locations, IDEs and how to compile/run in each language.
  • IDEs/Text-Editors
    • Atom
    • Code::Blocks
    • DrJava
    • Eclipse Cpp
    • Eclipse Java
    • Emacs
    • IDLE
    • IntelliJ IDEA
    • Notepad
    • Notepad++
    • PyCharm
    • Vim
    • Visual Studio Code
    • WordPad
